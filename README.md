LokoPizza
=========

Lokomotive vs. Pizza

----------------------------

Dies ist ein Projekt, das im Rahmen des [GameJam(0)](https://wiki.chaosdorf.de/GameJam(0)) entstanden ist, und sich mit `Pizza`, `Blocks`, `Trains` und `Michael Bay` beschäftigt.


Ausführen: `python main.py`
